# IH-Aging Multi-Omics Analysis Code

This repository contains analysis scripts supporting a manuscript investigating
aging-associated inguinal hernia through integrative single-nucleus multi-omics,
network analysis, statistical genetics, and spatial analyses.

## Code organization
All analysis scripts are provided in the `scripts/` directory.
Scripts are numbered to reflect the logical analysis workflow.

## Data availability
No raw or individual-level data are included in this repository.
All datasets used in the study are publicly available from their original sources as described in the manuscript.
