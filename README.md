# IH-Aging Multi-Omics Code

This repository contains analysis scripts supporting a manuscript on aging-associated inguinal hernia,
integrating single-cell multi-omics, network analysis, statistical genetics, and spatial analyses.

## Code
All scripts are provided in the `scripts/` folder and are numbered to reflect the analysis workflow.

## Data
No raw or individual-level data are included in this repository. Data sources are described in the manuscript.
